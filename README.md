# STATGEN 2026 Conference Website [https://statgen26.emory.edu/]

## Conference on Statistics in Genomics and Genetics

This repository contains the official website for STATGEN 2026, a conference organized by the ASA Section on Statistics in Genomics and Genetics.

### About the Conference

STATGEN 2026 will be held on **May 18-20, 2026** and aims to provide a forum for SSGG members and friends to present, connect, and foster ideas related to genomics, genetics and associated applications.
